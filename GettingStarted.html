Getting Started

Before we get started

For new and unfamiliar users

Once again, welcome to Computer Newbz: Computer Components and Assembly For Technologically Challenged.
Congratulations on taking your first step toward assembling your very own computer, which I'm sure won't
only be a very rewarding experience, but an enlightening one as well. You will find that building your own
computer is more cost-effective than buying a prebuilt computer with the EXACT same parts
one of the many compelling reasons for assembling your own computer.
However, before we disembark on this journey, here are a few things you MUST ALWAYS keep in mind:

  • Always handle the parts in a well lit, dust-free, static free environment
  • Make sure you always ground yourself before handling any computer components, if not, you run the
    The risk of short-circuiting them.
  • If using metal tools, like screwdrivers for instance, be very careful not to make any metal on metal
    contact with any of the components lest you risk damaging the hardware.
  • Handle the components with care, failure to do so could result in damage.
